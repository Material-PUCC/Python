Print command

Prints your desired text on screen. If using "sep=''", it will divide your text.

Using single quotes or double quotes may affect. When using text only, it will have no effect. If using variables, it is a must to include either quotes or double quotes to differentiate each of them.

For example:

print("Hello World")

will print "Hello World"

print('Hello World')

will print "Hello World"

print("Hello " + "World")

print(this, shows, how, sep, works, sep=" ")

will print "this shows how sep works"

Math Operations

print(2 + 2)

will print 4

print(11-3)

will print 8

print(15/3)

will print 5
Dividing will always result in a float/double value, due to it allowing some errors on bigger codes. For it not to show "leftovers", use '//'

print(17//5)

will print 3(note how it doesn't show the rest 2)

print(17%5)

will print 2(what's left from 17/5)

print(2**3)

will print 8

print(25**0.5)

will print 5

print(25**1/2)

will print 12.5
WARNING: Trying to use numbers such as '1/2' will not work, as it will execute the operations following the line's order. For it to work, use parenthesis.

print(25**(1/2))

will print 5

print(1+2)

will print 3

String Operations

print("1"+"2")

will print 12

 Executing an addition operation with string variables will concat them. print("PUC"+"-"+"Campinas")will print "PUC-Campinas"

You can use multiplication sign(*) on a print command with strings to print the string multiple times.

print("PUC "*3)

will print "PUC PUC PUC"
