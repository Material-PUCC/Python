Variables

Variables are used to store data. They can be named anything.

Variables can be predefined in the code or defined with an input

For example:

name = "John"

age = 20

print(name)

will print "John"

print(age)

will print 20

Another example:

name = input("What's your name?")

print("Hello, " + name)

will ask for your name and print "Hello, " + your name
